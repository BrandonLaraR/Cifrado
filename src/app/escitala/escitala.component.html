Cifrado Escitala

¿Qué es el cifrado Escitala?

El cifrado Escitala es un método de cifrado que utiliza una cinta enrollada en un cilindro para cifrar mensajes. Para descifrar el mensaje, se necesita el cilindro del mismo diámetro.

¿Cómo codificar con Escitala?

Coloca el mensaje en una cinta, que luego se enrolla alrededor del cilindro. El mensaje se leerá de arriba a abajo, formando una columna en el cilindro.

¿Cómo descifrar el cifrado Escitala?

Para descifrar el mensaje, desenrolla la cinta y lee el mensaje en la dirección correspondiente. Necesitas conocer el número de columnas utilizadas.

¿Cómo se reconoce el cifrado Escitala?

Un texto cifrado con Escitala puede ser identificado por su formato en columnas, que a menudo no se puede leer sin el cilindro adecuado.

Ingresa el mensaje que quieras cifrar o descifrar, así como el número de columnas para llevar a cabo el cifrado o descifrado.
